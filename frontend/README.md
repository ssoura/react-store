# Ecommerce platform using MERN Stack

## Usage

### backend
#### Set Environment Variables

- CD to backend
- Rename the .env.example to .env
- Set enviroment Variables

```
MONGO_URI =
JWT_SECRET =

```

#### Install dependencies and run

```bash
npm install
npm start
```
